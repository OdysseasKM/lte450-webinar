🎬 LTE450 Webinar - Embed Codes

Copy the code snippets below to integrate the webinar into the client's website.

Important: Replace /webinar/ with the actual path where you upload the webinar folder on the client's server.

1. Basic iframe Embed

Simple embed with fixed height. Works on most websites.

<iframe src="/webinar/index.html" width="100%" height="800" frameborder="0" allowfullscreen> </iframe>

2. Responsive iframe (16:9)

Maintains aspect ratio on all screen sizes. Best for modern responsive sites.

<div style="position: relative; width: 100%; padding-bottom: 56.25%; height: 0; overflow: hidden;"> <iframe src="/webinar/index.html" style="position: absolute; top: 0; left: 0; width: 100%; height: 100%; border: none; border-radius: 12px;" allowfullscreen> </iframe> </div>

3. Styled Embed with Shadow

Adds visual polish with rounded corners and shadow.

<div style="max-width: 1200px; margin: 40px auto; padding: 0 20px;"> <iframe src="/webinar/index.html" width="100%" height="750" style="border: none; border-radius: 16px; box-shadow: 0 8px 40px rgba(0,0,0,0.15);" allowfullscreen> </iframe> </div>

4. Full Page (Dedicated Webinar Page)

Makes the webinar fill the entire browser window.

<!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>LTE450 Webinar</title> <style> * { margin: 0; padding: 0; } html, body { height: 100%; overflow: hidden; } iframe { width: 100%; height: 100%; border: none; } </style> </head> <body> <iframe src="/webinar/index.html" allowfullscreen></iframe> </body> </html>

5. Direct Link (No Embed)

Simply link to the webinar page. Opens in same or new tab.

<!-- Same tab --> <a href="/webinar/index.html">Watch the LTE450 Webinar</a> <!-- New tab --> <a href="/webinar/index.html" target="_blank">Watch the LTE450 Webinar ↗</a>

6. Styled Button Link

Eye-catching call-to-action button.

<a href="/webinar/index.html" style="display: inline-block; background: linear-gradient(135deg, #d97706, #f59e0b); color: #1a1a2e; padding: 16px 32px; border-radius: 50px; text-decoration: none; font-weight: 700; font-size: 18px; box-shadow: 0 4px 20px rgba(245, 158, 11, 0.3);"> ▶ Watch the LTE450 Webinar </a>

7. WordPress (Gutenberg Block)

Add a "Custom HTML" block and paste this code:

<div class="webinar-container"> <iframe src="/wp-content/webinar/index.html" width="100%" height="800" frameborder="0" allowfullscreen loading="lazy"> </iframe> </div>
WordPress path: Upload the webinar folder to /wp-content/webinar/ via FTP or File Manager.

8. External Hosting (Netlify/Vercel)

If hosting the webinar separately, use the full URL:

<iframe src="https://lte450-webinar.netlify.app/" width="100%" height="800" frameborder="0" allowfullscreen> </iframe>
To deploy to Netlify:
1. Go to app.netlify.com/drop
2. Drag and drop the webinar folder
3. Get your free URL instantly
